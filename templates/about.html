This site is a (we hope) more useable version of the job listing site at careers.ottawa.ca, You can use this site to search the City of Ottawa career listings, which are updated daily and are always up-to-date with the ones found on the City website. The links to Apply to a listing will take you directly to the City of Ottawa application for that listing.

The site also makes the raw data available in JSON format and Eluta XML, to assist crawling with job aggregator sites:

  • JSON: http://www.ottawacityjobs.ca/en/data/

  • Eluta XML: http://www.ottawacityjobs.ca/eluta/

This site has been made possible because the City of Ottawa releases their job listings as Open Data. That data can be found here

The source code for this site can be found on Github

If you have any questions about the site, you can email us.
